How Java Works

You need to install a JDK (Java Development Kit) in order to write and execute Java programs. The JDK includes the JRE (Java Runtime Environment) which includes the JVM (Java Virtual Machine). The JVM works like an adapter that allows you to WORA (Write Once, Run Anywhere).
